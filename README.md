# Blue ver2 MVP

Blue ver2 is a small local-first health and habit tracker MVP.

## Features

- PIN entry screen with error state
- Daily body status inputs
- Today view with body summary and habit checklist
- Habit tracker with monthly and yearly dot grids
- Statistics view with body metrics chart and habit completion

## Run

Open `index.html` directly in a browser, or run:

```bash
npm start
```

Then open `http://localhost:5174`.

Default PIN: `0909`
